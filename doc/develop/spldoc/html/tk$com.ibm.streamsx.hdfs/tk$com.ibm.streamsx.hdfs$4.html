What is new in version 4.5.0

IBMStreams com.ibm.streamsx.hdfs Toolkit > com.ibm.streamsx.hdfs 4.5.0 > Developing and running applications that use the HDFS Toolkit > What is new in version 4.5.0

  • The streamsx.hdfs toolkit has been improved with a new operator HDFS2FileCopy.

  • HDFS2FileCopy copies files in to directions:

  • copyFromLocalFile : Copies a file from local disk to the HDFS file system. \n"
  • copyToLocalFile : Copies a file from HDFS file system to the local disk. \n\n"

  • Adapting of streamsx.hdfs to support HDP 2.x and HDP 3.x file systems.

  • The pom.xml file has been upgraded to use hadoop version 3.1.0 libraries.
